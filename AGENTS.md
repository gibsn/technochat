# Локальные инструкции для Codex

- Для этого проекта не запускай автоматически e2e и integration tests: они требуют поднятого окружения и могут конфликтовать с другой параллельной сессией. Если общий тестовый таргет включает e2e/integration, выбери безопасную более узкую проверку или явно пропусти эту часть.
